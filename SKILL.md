# 发票管理器

OCR识别PDF发票 → 按模板生成报销汇总 → 按行程归档。

## 目录

- `input/` — 用户放入待处理PDF发票
- `archive/` — 归档输出（汇总表 + 行程子文件夹）
- `templates/` — 报销模板 Excel

## 流程

### 1. OCR 识别

```bash
python3 scripts/process_invoices.py input/ --output /tmp/invoice_results.json
```

提取：金额、日期、费用类型、地点、销售方、发票号。

### 2. 确认信息

将结果发给用户确认，**必须问**：
- **归属项目** — 每次不同
- 费用类型、金额、地点是否正确
- 行程归类是否合理

### 3. 生成汇总表 + 归档

用 `scripts/generate_summary.py` 生成。需传入确认后的发票数据（JSON）。

**Excel 列映射：**
| 列 | 字段 | 来源 |
|---|---|---|
| B | 开始日期 | OCR |
| C | 结束日期 | OCR（同天=开始日期） |
| D | 出差地点 | OCR |
| E | 花费金额 | OCR（价税合计） |
| F | 费用类型 | OCR + 用户确认 |
| I | 归属项目 | 用户指定 |
| K | 文件上传 | 重命名后的PDF文件名 |
| L | 报销人 | 用户指定（默认贾睿童） |

**归档规则：**
- 一个月一个汇总表：`archive/{年月}_报销汇总.xlsx`
- 子文件夹：`archive/{年月}{出差地}{项目名称}差旅/`
- PDF 重命名：`{日期}_{费用类型}_{销售方简述}.pdf`
- 子文件夹内只放 PDF，不放汇总表

**费用类型选项：** 餐费、住宿费、差旅费用、市内交通费（含路桥费）、通讯费、信息服务费、快递费用、招待费/礼品费、其他费用

### 4. OCR 识别不准确时

用 `image` 工具直接查看 PDF 渲染图，手动补充信息。
